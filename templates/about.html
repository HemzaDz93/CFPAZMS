الصفحات والخصائص الرئيسية

  • لوحة التحكم: ملخص أداء النظام، بطاقات إحصائية وروابط سريعة للوظائف اليومية.
  • إدارة المخزون: إدارة الأصناف، التصنيفات، وسجل الحركات (وإمكانية جرد الأصناف).
  • الموردون: إدارة سجلات الموردين وأوامر الشراء المرتبطة.
  • المعدات: إدارة الأصول، تسجيل مشاكل/إصدارات، وتتبع حالات الأصول.
  • المطعم: إدارة الوصفات، الوجبات وسجل الاستهلاك.
  • طلبات الموظفين: واجهة إنشائية لانشاء ومتابعة طلبات الموارد الداخلية.
  • التقارير: تقارير حركة المخزون، الأصناف منخفضة المخزون، جرد الأصول وتقارير استهلاك المطعم.
  • ميزات متقدمة: مفاتيح API، تنبيهات ذكية، نظام الموافقات، ولوحة تحليلات.
  • الأمان: دعم تسجيل الدخول عبر OAuth (SSO)، مصادقة ثنائية (2FA)، وسجل نشاط المستخدم.
  • PWA و Offline: تدعم تخزين مؤقت للواجهات، مزامنة عند الاتصال، وإشعارات Push عبر VAPID.
  • التخصيص: إعدادات المؤسسة، إدارة المستخدمين والأذونات التفصيلية.

ملاحظات فنية

  • الواجهة مبنية باستخدام Flask وJinja2 وBootstrap (مع دعم RTL).
  • قاعدة البيانات تستخدم SQLAlchemy (افتراضياً SQLite أثناء التطوير).
  • بعض الميزات التحليلية تعتمد على مكتبات اختيارية (مثل scikit-learn أو Prophet). إذا لم تكن مثبتة، فسيتم تعطيل أجزاء التنبؤ تلقائياً دون التأثير على بقية النظام.
  • لتمكين الإشعارات Push، يجب إعداد مفاتيح VAPID في ملف البيئة `.env` أو متغيرات النظام.

كيفية المساهمة أو التطوير المحلي

  1. استنساخ المستودع وتشغيل بيئة افتراضية Python.
  2. تثبيت المتطلبات: pip install -r requirements.txt.
  3. تهيئة المتغيرات في `.env` (مفتاح VAPID، بيانات SSO إن وُجدت).
  4. تهيئة قاعدة البيانات: flask init-db.
  5. تشغيل الخادم في وضع التطوير: flask run.

معلومات إضافية

إذا رغبت بإضافة ميزات جديدة أو تحسينات (مثل دعم LDAP/Active Directory، جدولة مهام دورية، نسخ احتياطي تلقائي أو نظام إدارة مستندات متقدم) تواصل معي مع تفاصيل المتطلبات وسأقوم بمساعدتك في التخطيط والتنفيذ.
